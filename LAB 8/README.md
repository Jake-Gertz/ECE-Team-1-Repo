## Lab Questions

### In plain English describe the function and use of a multiplexer.

The function of a multiplexer is to filter multiple inputs into one single output. This is so you can send multiple input signals into a single channel and have them fill the same purpose. This allows you to route the data into a smaller signal, making it simpler and more effiecient.

### In plain English describe the function and use of a demultiplexer.

The function of a demultiplexer is to branch a single input into multiple outputs. The goal is to make the circuit more complex, and have more potential outputs. This takes the circuit from being less complex to being more complex, to potentially broaden the circuit and its uses.
